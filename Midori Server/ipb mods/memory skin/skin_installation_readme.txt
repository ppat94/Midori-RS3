Installation instructions.

1. Open your Admin Control Panel for IP.Board.

2. Go to the Look and Feel Tab and hover over Look and Feel > Import / Export and click that.

3. You will now be on the import page. Choose to import memory.xml

4. In the content box below, upload the image file: images-memory.xml

5. Choose to apply the image set to the skin you just imported.



That's it! You're done. For a guide with pictures, check here:
